Users App

Example Usage with cURL

Create a new user:

            curl -X POST -i http://localhost:4567/users -d "firstname=value1" -d "lastname=value2" -d "age=30" -d "password=123456" -d "email=user@example.com"

Retrieve all users:

            curl http://localhost:4567/users

Log in:

            curl -X POST -i http://localhost:4567/sign_in -d "email=user@example.com" -d "password=123456"

Update user's password:

            curl -X PUT -i http://localhost:4567/users -d "password=new_password"

Sign out:

            curl -X DELETE -i http://localhost:4567/sign_out

Delete user:

            curl -X DELETE -i http://localhost:4567/users
